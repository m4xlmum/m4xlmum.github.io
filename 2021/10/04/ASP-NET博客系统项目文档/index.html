ASP.NET博客系统项目文档

前言

想做这个东西很久了，主要是想尝试一下MVC的网站开发模式，看看网站从无到有的开发的流程。今天开始可以开始自己实现一个了，我会把开发的过程记在这个文档里面，留作以后的参考。

注:

  1. 本系统暂时只会实现个人发表博客，登录功能主要是用于发表文章更新时用的，相当于一个后台的admin管理界面。

一、前端布局

这里我主要想参考的是下面这个hexo的博客的布局来搞，就是一个初级的练手的项目，不想搞得太深，没什么必要也。

就是最上面是一个导航栏，再往下一点有个轮播图，再向下是个三栏式的博客，左边和右边是小工具的地方，中间放文章的标题与摘要，可以设置封面，不设置封面的话自动加载默认的图片。

二、数据库设置

两个表，一个表中存放用户的相关信息，另一个表中存放发表的博客文章的相关信息。

1. 用户信息表

用户名、密码、邮箱

CREATE TABLE [dbo].[userinfo]
(
	[Id] INT NOT NULL PRIMARY KEY, 
    [username] VARCHAR(10) NOT NULL, 
    [password] VARCHAR(20) NOT NULL, 
    [email] VARCHAR(50) NOT NULL
)

2. 文章相关表

文章id(唯一)、文章cover、文章标题、文章路径(并不准备把文章存在数据库里面)。

CREATE TABLE [dbo].[ArticleInfo] (
    [Id]             INT            NOT NULL,
    [articlecover]   NVARCHAR (200) COLLATE SQL_Latin1_General_CP1_CI_AS NULL,
    [articletitle]   NVARCHAR (50)  COLLATE SQL_Latin1_General_CP1_CI_AS NOT NULL,
    [articlepath]    NVARCHAR (200) COLLATE SQL_Latin1_General_CP1_CI_AS NOT NULL,
    [articlesummary] NVARCHAR (200) COLLATE SQL_Latin1_General_CP1_CI_AS NOT NULL,
    [articlewordnum] INT            NOT NULL DEFAULT 0,
    [articlesticky]  INT            NOT NULL DEFAULT 0,
    [articleupdatetime] DATETIME NULL, 
    [articlelike] INT NOT NULL DEFAULT 0, 
    CONSTRAINT [PK_dbo.ArticleInfo] PRIMARY KEY CLUSTERED ([Id] ASC)
);

文章标题不能包含#、，以及windows命名文件时不允许出现的字符

1. 登录判定(确认密码)

public ActionResult Login(string username, string password)
{
    string truepass = "";
    foreach (var pass in db.userinfo.Where(user => user.username == username))
    {
        truepass = pass.password;
    }
    if (password == truepass)
    {
        string page = Url.Action("Index", "Admin");
        return Redirect(page);
    }
    else
    {
        ViewBag.IsLoginError = true;
        return View();
    }

}

这里设置了一个ViewBag.IsLoginError的值，代表的是是否登陆失败，true表示登陆失败，当登录失败时，html上面做出一个提示，代码如下。

Controller

[HttpPost]
public ActionResult Register([Bind(Include = "username,password,email")] userinfo userinfo)
{
    if (ModelState.IsValid)
    {
        userinfo.Id = db.userinfo.Count();
        //验证username的唯一性，如果username已存在则不给添加
        foreach (var name in db.userinfo.Where(user => user.username == userinfo.username))
        {
            ViewBag.IsRegistered = true;
            return View(userinfo);
        }

        db.userinfo.Add(userinfo);
        db.SaveChanges();
        return RedirectToAction("Login");
    }

    return View(userinfo);
}

2. C#有字符串生成MD5

using System.Text;
using System.Security.Cryptography;

/// <summary>
/// MD5字符串加密
/// </summary>
/// <param name="txt"></param>
/// <returns>加密后字符串</returns>
public static string GenerateMD5(string txt)
{
    using (MD5 mi = MD5.Create())
    {
        byte[] buffer = Encoding.Default.GetBytes(txt);
        //开始加密
        byte[] newBuffer = mi.ComputeHash(buffer);
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < newBuffer.Length; i++)
        {
            sb.Append(newBuffer[i].ToString("x2"));
        }
        return sb.ToString();
    }
}

using (BlogDb tmpdb = new BlogDb()) {

    path = Request.MapPath(path);  //转换为相对路径
